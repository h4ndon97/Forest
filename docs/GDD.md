## 2. 세계관

> 상세: STORY.md

### 배경
원래 하나였던 왕가가 빛의 계보(현재 왕가)와 그림자의 계보(그림자 왕가)로 분열되었다. 현재 왕가는 제사장 가문을 이익(지위와 능력)으로 포섭하여 그림자 왕가를 숲에 봉인했다. 오랜 시간이 지나 현재 왕가가 봉인된 힘에 손을 댔고, 봉인이 균열을 일으키며 저주가 확산되기 시작했다. 제사장 가문의 선조가 시간을 멈춰 확산을 봉쇄했으나, 진실은 묻혔다.

현재: 시간이 멈춘 숲. 그림자 왕가의 분노와 귀환 의지가 스며들어 오브젝트마다 그림자 괴물이 생성되고 있다.

### 주인공
숲속 왕국 제사장 가문의 후예. 선조를 영웅으로 알고 저주 해결을 위해 숲에 진입한다. 탐색을 거치며 선조의 협력이 이익을 위한 선택이었음을, 그리고 자신이 물려받은 능력(시간 조작, 등불)이 그 거래의 산물임을 발견하게 된다.

### 저주의 정체
그림자 왕가의 분노와 귀환 의지. 근본 원인은 두 계보가 부당하게 분리·봉인된 것이며, 현재 왕가가 봉인된 힘에 손을 댄 것이 방아쇠가 됐다.

### 최종 목표
분노와 뒤틀림 속에 불완전하게 결합된 두 왕가(최종 보스)를, 빛과 그림자를 모두 다루어 올바른 방식으로 해방·정화한다.

### 핵심 테마
"물려받은 것의 무게" — 주인공이 물려받은 능력은 과거의 부당한 거래에서 비롯되었으며, 그것을 인지하고 바로잡는 여정.

## 3. 핵심 메카닉

> 상세: CORE_SYSTEMS.md

### 3.1 낮/밤 사이클
- 경계: 06:00 (일출), 18:00 (일몰)
- 직접 낮↔밤 전환 불가 — 시간 흐름으로만 전환
- 조작 가능 범위: 경계 10분 전까지 (05:50, 17:50)
- **디폴트: 시간 정지**

### 두 가지 시간 행동

| 구분 | 트리거 | 속도 | 자원 | 적 상태 |
|---|---|---|---|---|
| **시간 조작** | 버튼 꾹 누름 | 빠름 (연출) | 소모 | 없음 |
| **시간 흐름** | 조작 후 해제 | 1분 = 1시간 | 회복 | 활성화 |

- 시간 조작: 멈춘 상태에서만 가능. 원하는 시각까지 빠르게 이동. 자원 소모.
- 시간 흐름: 적 활성화. 자원 회복. 자원 없이도 항상 가능 → 데드락 없음.

### 3.2 그림자 크기 ↔ 적 강도
- 그림자 크기가 적 강도를 결정 — **연속적 연동** (20%~150% 통합 배율)
- HP, 공격력, 속도를 동시에 조정

| 시간대 | 그림자 짧음 | 그림자 김 |
|---|---|---|
| **낮** | 20% (최약) | 150% (최강) |
| **밤** | 150% (최강) | 20% (최약) |

- 낮: 정오(그림자 최소) → 20%, 경계 근처(그림자 최대) → 150%
- 밤: 등불 가까이(그림자 짧음) → 150%, 등불 멀리(그림자 김) → 20%
- **구현**: 스프라이트 기반 (Light2D/Shadow2D 미사용). 시간 변수에 따라 offset/scale 조절.

### 3.3 등불
- 영구 도구 (장비가 아님, 성장 시스템으로 강화)
- 밤에 사물에 빛을 비춰 그림자를 생성 → 적 소환 및 처치 가능
- 게임 진행으로 강화 (밝기, 범위)

### 3.4 시간 자원
- 시간 조작 시 소모, 시간 정지 상태에서는 소모/회복 없음
- **빡빡한 자원** — 매 스테이지를 최적 조건으로 싸우는 것은 불가능
- 자원 부족 = 불리한 조건으로 전투하는 페널티 (데드락 없음)

| 회복 수단 | 조건 | 성격 |
|---|---|---|
| 시간 흐름 자동 회복 | 시간이 흐르는 중 (적 활성화) | 기본. 리스크 감수 필요 |
| 아이템/소모품 | 탐색 보상으로 획득 | 비상용 |
| 적 처치 시 회복 | 적 처치 | 공격적 플레이 장려 |

### 3.5 땅거미

> 상세: CORE_SYSTEMS.md §4

- 시간이 흐르는 맵을 감지하고 찾아오는 특수 적
- **복수 존재** — 월드맵에 여러 마리 동시 존재
- **이동 속도**: 맵 1개 = 실시간 2분 (거리 기반)
- 도착 시: 적 강화 + 잔류 부활 + 전투 적으로 등장 (3중 위협)
- **HUD 접근 경고**: 2맵 이내 접근 시 화면 우상단 보라색 맥동, 1맵 시 긴급 경고, 도착 시 플래시 연출
- 대응: 시간 정지로 감지 차단, 또는 직접 처치

## 4. 전투 시스템

> 상세: COMBAT.md

### 4.1 전투 수단 3가지

| 수단 | 설명 | 역할 |
|---|---|---|
| **직접 공격** | 기본 공격 + 스킬 | 액션 레이어 |
| **그림자 약화** | 시간/빛 조절로 적 약화 | 매크로 조작 |
| **환경 활용** | 환경 오브젝트로 정밀 조작 | 마이크로 조작 |

### 4.2 기본 공격
- **무기**: 검 (근접). 검 교체 가능, 검마다 다른 능력치/특성.
- **4타 콤보** — 횡베기 (좌우)
  - 1~3타: 무속성 횡베기
  - 4타(피니시): 최다 투자 스킬 트리의 속성 공격 (빛/그림자/혼합)
- 자원 소모 없음

### 4.3 이동 체계
- **처음부터 사용 가능**: 달리기 (단일 속도), 2단 점프, 대시
- **강화 이동**: 진행에 따라 해금. 빛/그림자 테마 연결 (빛 대시, 그림자 점프 등). 스테이지 잠금 통과에 활용.

### 4.4 환경 오브젝트

| 오브젝트 | 효과 | 전투 활용 | 구현 |
|---|---|---|---|
| 거울/수정 | 빛 분산 → 그림자 분열 | 강한 적 → 약한 적(그림자 파편) 다수로 분할 | **Phase 2-5a 완료** |
| 렌즈(볼록) | 빛 집중 → 특정 그림자 극도로 축소 | 다수 중 하나만 골라 약화 | **Phase 2-5c 완료** |
| 차폐물(기둥, 벽) | 빛 차단 → 그림자 강제 생성/제거 | 투영 영역 내 적 강도 override | **Phase 2-5b 완료** |
| 반사 바닥(물, 얼음) | 아래에서도 빛 → 이중 약화 | 특정 지형에서 유리한 전투 | **Phase 2-5d 완료** |

- **수동 조작** — 시간 정지 상태에서만 가능, 흐름 중에는 고정
- "세팅(정지) → 실행(흐름)" 리듬
- **거울 동작 (Phase 2-5a)**: STOPPED 중 E키로 4프리셋(90°씩) 회전 → FLOWING 진입 시 부채꼴 영향권(60°/128px) 내 적이 '그림자 파편' 스포어 다수로 분열. 부활체/스포어는 재분열 가드로 무시.
- 분열 인프라는 적의 사망 분열(`death_behavior_split`)과 공용(`split_spawner.gd`). `BaseEnemy.trigger_split()` public API로 외부에서 트리거.
- **차폐물 동작 (Phase 2-5b)**: STOPPED 중 E키로 플레이어 반대 방향 1스텝(±16px, ±64px 클램프) 밀기. 본체 뒤쪽 직사각형 투영 영역(96×32, 낮=태양 반대/밤=등불 반대)에 들어간 적의 그림자 강도를 `block_intensity(0.9)`로 상시 강제 override(max 병합). 이탈 시 낮/밤 상태에 맞는 원래 강도로 복원. CREATE 모드만 구현(REMOVE/BOTH는 자리 확보).

### 4.5 피격 / 생존
- **HP**: 수치형 — 그림자 크기에 따른 데미지 차이가 직접 체감됨
- **회복**: 아이템(전투 중) + 자동 회복(시간 정지 상태에서 1초 간격 2.0 HP, Phase 2-1 구현)
- **사망**: 거점 귀환, 페널티 없음

## 5. 스킬 시스템

> 상세: SKILLS.md

### 5.1 세 갈래 스킬 경로

| 계열 | 공격 성격 |
|---|---|
| **빛** | 빠르고 날카로운 공격. 직선적 공세. |
| **그림자** | 묵직하고 넓은 공격. 범위 공세. |
| **혼합** | 빛+그림자 융합 공격 + 연쇄 콤보 보너스 |

- 세 갈래 모두 투자 가능, **포인트 한정** → 비율 선택
- 혼합 해금: 빛과 그림자에 각각 일정 이상 투자 필요
- 기본 공격 + 이동공격 기반, 복잡한 기믹보다 **액션감** 중시

### 5.2 슬롯 구조
- **기본 공격** 1개 (고정)
- **스킬 슬롯** 4개 (자유 장착, 트리 제한 없음, 쿨타임 기반)
- 트리당 6개 스킬 → **총 18개**

### 5.3 잠금 해제 능력
- 스킬 포인트와 **별도** 획득 (보스 처치, 탐색 보상 등)
- 스킬 포인트 = 순수 전투 성장

## 6. 적 시스템

> 상세: ENEMIES.md

### 6.1 그림자 적
- 오브젝트의 그림자에서 생성, 시간이 흐르면 등장
- 원본 오브젝트의 형태 → 적의 행동 패턴 결정

### 베이스 4종

| 오브젝트 | 특성 | 그림자 반응 |
|---|---|---|
| 나무 | 크고 느림, 범위 넓음 | 표준 (20%~150%) |
| 바위 | 작고 단단함 | 둔감 (40%~150%) |
| 꽃 | 빠르고 약함, 분열 | 민감 (10%~150%) |
| 돌기둥 | 길고 가늘음, 원거리 | 표준 (20%~150%) |

### 구현 상태 (Phase 1-5 완료)
- **EnemySystem**: Autoload로 적 활성화/비활성화, 강도 갱신, 등록/제거 관리
- **베이스 4종**: 나무/바위/꽃/돌기둥 데이터 리소스 완료 (`data/enemies/`)
- **그림자 강도 ↔ 적 스탯**: 실시간 연동 (유형별 min/max 재매핑)
- **AI**: 7상태 상태 머신 (DORMANT~DEAD)
- **임시 전투**: 플레이어 공격/피격 브릿지 (CombatSystem 전까지)

### 6.2 구역별 확장
- **서브 타입**: 같은 오브젝트지만 외형/패턴이 다른 거의 별개의 적 (구역마다 등장)
- **구역 고유 적**: 해당 구역에만 존재하는 완전 신규 적 (베이스 4종과 무관)

### 6.3 그림자 잔류
- 처치한 적의 잔여 그림자가 맵에 남음
- 땅거미 도착 시 부활: **HP 50%, 공격력 150%** (유리대포)
- 정화 능력(중반 해금)으로 완전 제거 가능

## 7. 보스전 설계

> 상세: BOSSES.md

### 7.1 보스 정체성
- **초반 (1~2구역)**: 숲의 거대 오브젝트가 괴물화. 단순한 저주의 산물.
- **중반 (3~4구역)**: 괴물화 + 그림자 왕가의 잔재 결합. 사연이 느껴지기 시작.
- **최종 (5구역)**: 두 왕가의 불완전한 결합체.
- 보스 진행 = 스토리 진행.

### 7.2 전투 구조
- 매크로(시간 조절) + 마이크로(환경 조작) + 액션(직접 공격) 복합 사용
- **멀티 페이즈** — HP 구간마다 패턴 변화

| 구역 | 페이즈 |
|---|---|
| 1~2구역 | 2 |
| 3~4구역 | 3 |
| 5구역 | 4 |

### 7.3 보스 처치 보상 (3가지 동시)

| 보상 | 역할 | 예시 |
|---|---|---|
| 잠금 해제 능력 | 게임플레이 확장 | 빛 대시, 그림자 점프 등 |
| 스토리 단서 | 내러티브 진행 | 컷신, 기록물 |
| 강화 보상 | 성장 | 대량 강화 포인트, 고유 장비/장신구 |

- 다음 구역 해금 기본 포함

## 8. 맵 / 스테이지 시스템

> 상세: STAGES.md

### 8.1 맵 등급

| 등급 | 역할 | 특징 |
|---|---|---|
| **일반 맵** | 탐색/전투 | 인접 포탈로만 이동 |
| **거점 맵** | 중심지/허브 | 월드맵 포탈, 세이브, 완전 회복, 강화/장비 관리 |

- 거점은 **구역 경계에 위치**, 네트워크형 연결
- **마을형** (NPC, 스토리) 3개 + **간이형** (오브젝트만) 5개 = 총 8개

### 8.2 구역 및 스테이지 구성

| 구역 | 스테이지 | 보스 |
|---|---|---|
| 1구역 | 6 | 1 |
| 2구역 | 8 | 1 |
| 3구역 | 10 | 1 |
| 4구역 | 12 | 1 |
| 5구역 | 14 | 1 |
| **합계** | **50 + 보스 5** | |

### 8.3 클리어 구조

| 단계 | 상태 | 설명 |
|---|---|---|
| 미클리어 | 오브젝트 적 존재 | 진행 중 |
| 절반 클리어 | 적 전부 처치, 잔류 남음 | 땅거미 도착 시 부활 위험 |
| 완전 클리어 | 잔류까지 제거 | 안전 구역 확보 |

- 구역 보스 해금: 해당 구역 모든 스테이지 **완전 클리어**
- **잔류 정화 조건**: 처치한 시간대의 반대 시간대에 등불 접촉 (낮 처치 → 밤 정화, 밤 처치 → 낮 정화)

### 구현 상태 (Phase 2-8b 완료)

## 14. 진행 루프

### 맵 내 핵심 루프
```
맵 진입 (시간 멈춤 상태)
  │
  ├─ 현재 시각 확인 / 환경 오브젝트 배치 (세팅 페이즈)
  │
  ├─ 시간 자원 소모 → 시간 조작으로 원하는 시각 설정
  │
  ├─ 시간 흐름 → 적 활성화 → 전투 수행 (A/B/C 조합)
  │
  ├─ 동시에 땅거미 접근 중
  │   └─ 도착 전 시간 정지 or 땅거미째 처치
  │
  └─ 시간 흐름 중 자원 자동 회복 (리스크-리워드)
```

### 전체 진행 루프
```
초반: 적 처치 중심 (절반 클리어가 최선)
  │
  ├─ 능력 해금 (정화 능력, 강화 이동 등)
  │
중반: 정화 능력 → 이전 맵 재방문 → 완전 클리어 + 추가조작 발견
  │
  ├─ 환경 활용 능력 확장 → 새 경로 개방
  │
후반: 저주의 근원과 대면 → 최종 보스
```

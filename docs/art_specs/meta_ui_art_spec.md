# 시간이 멈춘 숲 — 메타 UI 아트 명세서

> 인게임 HUD가 아닌 **"게임 외곽 UI"** 전반을 다룬다.
> 타이틀 / 메뉴 / 세이브 / 설정 / 로딩 / 인벤토리 프레임 / 대화 / 버튼 프롬프트 / 커서.
> Phase 3 후반~Phase 5 사이에 제작되지만, 아트 방향성은 Phase 3-7 1구역 아트 확정과 동시에 결정되어야 한다.

---

## 0. 공통 규정

### 해상도 / 스케일
- 인게임 기본 해상도: 640×360 → 3× 스케일링 → 1080p
- 메타 UI도 **640×360 기준 좌표계**로 설계
- 단, 텍스트 가독성을 위해 **UI 아트 에셋은 2배 해상도(1280×720)로 제작 후 축소 표시** 권장 → 텍스트/아이콘 선예도 유지
- 픽셀아트 원칙은 유지하되, UI 프레임/배경은 2배 해상도 허용

### 폰트 정책 (미결 — Task 3 리서치에서 결정)
- 기본 UI 폰트는 **한글 지원 픽셀 폰트** 필요
- 영문만 쓰는 비트맵 폰트는 채택 불가 (한국어 게임이므로)
- 제목/로고는 별도 비트맵 이미지로 처리 (폰트 무관)

### 컬러 팔레트
- 메타 UI 전용 팔레트는 **인게임 팔레트와 조화** 필요
- 주요 톤:
  - 배경: `#1A1410` (어두운 숲 갈색)
  - 프레임: `#5A4A30` (낡은 가죽/나무)
  - 강조: `#F2CC66` (등불 금색) / `#8B2FC6` (그림자 보라)
  - 텍스트 기본: `#F0E8D8` (양피지 크림색)
  - 텍스트 비활성: `#8A8070`
  - 위험/경고: `#C84A4A`

### 스타일 톤
- **"낡은 양피지 책" 느낌** — 가죽 장정, 구겨진 종이, 잉크로 손글씨 쓴 듯한 느낌
- 과한 장식 금지. 여백과 대비 활용
- 애니메이션은 최소화 — 호흡감 주는 미묘한 펄스 정도

### 1-3. 제작 가이드

**배경 컨셉**:
- 플레이어 캐릭터 뒷모습이 숲 입구에 서 있는 구도
- 오른쪽 상단 멈춘 달/태양 → 게임 세계관의 시간 정지 암시
- 전경에 선명한 실루엣, 중경은 안개, 원경은 어두운 숲
- 움직임은 앰비언트 파티클(반딧불)로만

**로고 제작**:
- 한글 "시간이 멈춘 숲" → 5자. 두꺼운 획 + 중간중간 깨진 느낌
- 색상: 금색 #F2CC66 → 보라 #8B2FC6 그라데이션 (시간 정지 메카닉 암시)
- 가장자리 미세한 글로우 1px
- 비트맵 이미지이므로 폰트 선택에서 제외 → 픽셀 단위로 직접 디자인

**미결**:
- 영문 부제 확정 (현재 후보: "Forest of Stilled Time" / "Where Time Sleeps")
- 로고 발광 애니메이션 유무

### 2-3. 제작 가이드

**9-slice 구성**:
```
+----+----+----+
| 코 | 상 | 코 |   ← 16x16 + 16x16 타일링 + 16x16
+----+----+----+
| 좌 | 중 | 우 |   ← 타일링 가능해야 함
+----+----+----+
| 코 | 하 | 코 |
+----+----+----+
```
중앙 타일은 반투명 배경 (알파 0.85) + 미세한 양피지 텍스처.

**선택 강조**:
- 현재 호버/선택된 행의 배경. 반투명 금색 #F2CC66 (알파 0.3)
- 4프레임 루프로 아주 미세하게 맥동

**포인터**:
- 작은 삼각형 또는 화살표. 선택 행 왼쪽에 배치
- 2프레임 — 정지 / 오른쪽으로 1px 이동 (팅글)

## 12. 9-slice 공통 제작 가이드

여러 섹션에서 9-slice 프레임을 사용한다. 통일된 제작 규격:

```
┌────┬────────┬────┐
│ TL │   T    │ TR │   ← 모서리 16x16, 상변 타일링
├────┼────────┼────┤
│ L  │   C    │ R  │   ← 좌/우 타일링, 중앙 배경
├────┼────────┼────┤
│ BL │   B    │ BR │
└────┴────────┴────┘
```

**제작 규칙**:
- 각 타일은 16×16 또는 24×24 (섹션에 따라)
- 모서리(TL/TR/BL/BR)는 장식 포함 가능
- 변(T/B/L/R)은 타일링 가능해야 함 (이음새 없음)
- 중앙(C)은 반투명 배경 (알파 0.85) + 미세 텍스처 (선택)

**Godot 적용**:
- `NinePatchRect` 노드 사용
- `patch_margin_left/top/right/bottom` 에 타일 크기(16 또는 24) 설정

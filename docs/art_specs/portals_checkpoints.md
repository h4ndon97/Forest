## 2. 스테이지 포탈 (아치형 벽돌 + 덩굴)

**위치**: 맵 경계 (좌/우).
**크기**: 48×75 (폭 × 높이). Node2D 원점 중심.
**충돌**: 48×80 직사각형 (원점 중심, 살짝 여유).
**모티프**: 숲 속에 방치된 고대 아치 유적 + 자라난 덩굴. 캐릭터보다 약간 큰 규모(키 2배).

### 구조
- **기둥**: 폭 8 × 높이 50, 좌우 대칭. 원점 기준 x=±24~±16.
- **아치 상단**: 중심 (0, Y_PILLAR_TOP), 반경 20, 두께 8의 반원 링 (위쪽 반).
- **개구부(포탈 내부)**: 기둥 사이 + 아치 내부 전체.

### 레이어 (아래→위)
1. **포탈 내부 발광**: 개구부 중앙 3겹 원형 그라데이션 (반경 18/12/6, 금색→아이보리 코어)
2. **기둥 본체**: 석재 베이스 `#6B6560`, 5단 벽돌 (가로 모르타르 라인)
3. **아치 링**: 두께 8 반원, 5개 방사형 모르타르 라인 (30° 간격)
4. **석재 가장자리**: 좌측 1px 하이라이트 / 우측 1px 그림자 + 아치 좌상단 하이라이트
5. **덩굴**: 기둥 외측 2줄 + 아치 바깥 1줄 (폭 1.5, 사인 웨이브 흔들림)
6. **잎사귀**: 덩굴 위에 작은 원 (반지름 1.8) 배치 — 기둥 4개/아치 4개

### 애니메이션
- 내부 발광 펄스: 주기 2.4s, 강도 0.7↔1.0
- 플레이어 근접: 발광 강도 ×1.4 부스트 (lerp 속도 3.0/s)

### 상호작용 힌트
- 플레이어 영역 진입 시 포탈 내부 발광이 밝아짐 (부스트 1.4배)
- 장식 덩굴/잎은 정적 — 시간 정지 컨셉과 일관

## 3. 월드맵 포탈 (거점 중앙)

**위치**: 거점 중앙 (320, 304).
**크기**: 48×48.
**모티프**: §2-7 F-10 "나무 심벌" + §F-4 "수액 흐르기".

### 기하
- 줄기: 폭 6, 높이 20, 나무판 브라운 중앙 수직
- 뿌리: 3갈래, 하단에서 바깥/아래로 뻗음 (길이 8, 두께 2)
- 잎사귀: 5개, 줄기 양옆/상단에 삼각형 배치, 새순 연녹
- 후광: 반경 22, 금색 α=0.2 원형 그라데이션

### 애니메이션
- 후광 펄스: 주기 1.8s, α=0.15↔0.25
- 플레이어 근접 시 잎사귀 연녹→금색 전환 (0.3s lerp)
- 활성 시 잎 끝에 반딧불 점 2~3개 잠시 나타남

### 라벨
- 라벨: "[F] 월드맵" (HudLabel 테마 변형)
- 위치: 포탈 상단 -40px

## 4. 거점 세이브 마커 (RestMarker)

**위치**: 거점 중앙 휴식 지점 (320, 308).
**크기**: 48×40.
**모티프**: §2-7 F-10 "나무 심벌" (거점 버전 — 월드맵 포탈보다 더 안정적 존재감).

### 기하
- 땅 베이스: 타원 12×3, 나무판 브라운 α=0.6 (뿌리가 박힌 땅 그림자)
- 줄기: 폭 5, 높이 24, 나무판 브라운, 아랫쪽 살짝 두꺼워짐(2px)
- 큰 가지 2개: 좌우로 대칭, 각 길이 10 / 각도 -35°, +35°
- 잎사귀: 7개 (상단 5 + 측면 2), 새순 연녹
- 잎 하이라이트: 각 잎 위쪽 1px 아이보리 점

### 애니메이션
- 정적 기본 (미세 흔들림 없음 — 안정감)
- 거점 완전 회복 이벤트 시: 잎사귀 전체 금↔시안 2회 플래시 (A-10 규격)
- 앰비언트: 상단 0.3Hz 반딧불 점 1개 (선택적)

### 상호작용
- 플레이어 영역 진입 시: 줄기 밑동에 미세 발광 (α=0.4 금색)
- 세이브 완료 시: 잎사귀 순차 점등 + 포자 확산(F-05) 1회

## 5. 교체 워크플로

### 스프라이트 교체
1. **PNG 제작**: 각 요소의 크기 그대로 (48×75 / 48×48 / 48×40)
2. **임포트 설정**: 필터 Nearest, Mipmap 끔
3. **리소스 배치**: `assets/sprites/objects/{portal_stage.png, portal_world_map.png, rest_marker.png}`
4. **자동 반영**: 각 visual 스크립트의 `sprite_path` export에 경로 지정. 있으면 Sprite2D로 렌더, 프로그래밍 드로잉 자동 숨김.

### 픽셀아트 팁
- **스테이지 포탈(아치)**: 48×75. 기둥 1칸은 8×10 벽돌 5단, 아치는 약 13개 쐐기돌. 덩굴은 1px 두께로 기둥 바깥 타고 오르며 잎은 2×2. 내부 발광은 반투명 원 2~3겹
- **나무 심벌**: 48×48은 작다. 잎 1개를 3×3 픽셀로 시작, 줄기는 3×20이면 충분
- **애니메이션**: 정적 PNG 교체 시 애니메이션 로직은 그대로 남음 (Sprite2D modulate에 적용)

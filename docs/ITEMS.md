# 시간이 멈춘 숲 — 7. 아이템 시스템

> 이 문서는 장비, 장신구, 소모품의 종류와 획득 경로를 정의한다.
> 성장 시스템(GROWTH.md)과 전투 시스템(COMBAT.md)에서 확정된 구조 위에서 설계된다.

---

## 1. 아이템 카테고리

### 확정된 규칙 (STAGES.md, GROWTH.md, COMBAT.md)
- 장비/장신구를 통한 캐릭터 강화
- 획득 경로: 전투(적 처치 드롭), 맵 상호작용, 기믹/퍼즐 풀이, 보스 처치
- 프로퍼티 능력치(시간 자원 등)는 장비로도 올릴 수 있음
- HP 회복 아이템 존재 (전투 중 사용)
- 시간 자원 회복 아이템 존재

### 카테고리 (확정)
- **장비**: 무기 (1슬롯) + 방어구 (1슬롯)
- **장신구**: 3슬롯
- **소모품**: 일회성 사용 아이템
- **등불**: 장비가 아님. 성장 시스템으로 강화.

총 장착 슬롯: **무기 1 + 방어구 1 + 장신구 3 = 5슬롯**

---

## 2. 장비

### 무기 (확정)
- 검 교체 가능
- 검마다 다른 능력치/특성

### 방어구 (확정)
- 슬롯 1개

### 미결 사항
- [ ] 장비 등급 체계 여부
- [ ] 장비 강화 시스템 여부

---

## 3. 장신구

### 구조 (확정)
- 슬롯 3개

### 효과 방향 (확정)
- **능력치 보정 + 특수 효과** — 장신구마다 다름
  - 일부: 수치 보정 (공격력 +N, HP +N 등)
  - 일부: 고유 효과 (피격 시 그림자 폭발, 콤보 피니시 범위 증가 등)

### 미결 사항
- [ ] 장신구 조합/시너지 여부

---

## 4. 소모품

### 소지 구조 (확정)
- **종류별 개수 제한** — 종류마다 최대 소지량 설정
- 거점에서 보충 가능

### 미결 사항
- [ ] 소모품 종류 (HP 회복, 시간 자원 회복, 버프 등)
- [ ] 종류별 최대 소지량 (밸런싱)
- [ ] 획득 경로 (드롭? 상점? 제작?)

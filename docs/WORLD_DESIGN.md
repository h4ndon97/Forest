# 시간이 멈춘 숲 — 5. 월드 디자인

> 이 문서는 구역별 테마, 세계 배치, 거점 배치, 추가조작 유형을 정의한다.
> 스테이지 시스템(STAGES.md)에서 확정된 구조 위에서 설계된다.

> **현재 구현 상태 (2026-04-18)**: 테스트 맵 5개(test_stage ~ test_stage_5) + 거점 2개(test_checkpoint, test_checkpoint_2)로 스테이지 시스템 · 시간 전파 · 월드맵 프레임워크 검증 완료. 1구역 본격 레벨 디자인은 **Phase 3-1**에서 착수. 이하 구역 테마·세계 배치·거점 배치·추가조작 유형은 모두 설계 초안이며 각 구역 Phase 진입 시 확정한다.

## 1. 구역별 테마

### 확정된 구조
- 5구역, 총 50스테이지 + 보스 5개
- 선형 진행 (이전 구역 보스 처치 → 다음 구역 해금)
- 구역이 깊어질수록 그림자 왕가의 흔적이 강해짐 (스토리 연동)

### 구역별 테마 (초안)

전체 방향: **몽환적 → 점점 어두워짐**

| 구역 | 테마 | 분위기 | 스토리 연동 |
|---|---|---|---|
| **1구역** | 빛의 숲 | 몽환적, 빛이 나뭇잎 사이로 쏟아짐. 아름답지만 그림자가 꿈틀거림. | 표면적 저주. 메카닉 학습. |
| **2구역** | 안개 습지 | 안개 자욱한 늪지대. 시야 흐림, 물 반사면 존재. 불안함. | 숲이 뭔가 숨기고 있다는 느낌. |
| **3구역** | 폐허의 숲 | 고목과 옛 왕국 유적이 뒤섞임. 인공물 등장. | 전환점 — 그림자 왕가 흔적 발견. |
| **4구역** | 그림자 숲 | 빛이 거의 들지 않는 깊은 숲. 수정/광물 지형. 압도적 어둠. | 그림자 왕가의 영역. 진실 드러남. |
| **5구역** | 봉인의 심부 | 왕좌/제단이 있는 가장 깊은 곳. 빛과 그림자가 뒤틀려 공존. | 저주의 근원. 최종 대면. |

## 2. 세계 배치

### 확정된 규칙
- 월드맵: 하나의 연속된 숲을 탑뷰로 조감
- 자연 경계(강, 절벽, 바위벽 등)로 영역 구분
- 모든 거점은 구역 경계에 위치, 네트워크형 연결
- 구역 간 진입 통로 복수

### 배치 형태 (확정)
- **나선형** — 숲 바깥에서 안쪽으로 파고들어가는 구조
- 후반 구역은 넓이가 아니라 **밀도와 깊이**로 스테이지 수 증가
  - 수직 확장 (지하, 다층 구조)
  - 밀도 증가 (갈림길, 숨겨진 경로)
  - 복잡도 상승 (스테이지 내부 구조)
- 월드맵에서는 안쪽이 작아 보이지만, 실제로는 깊이가 있음
- "숲은 안으로 갈수록 깊어진다"

```
1구역(입구) → 2구역
                    ↓
 5구역(심부) ← 4구역 ← 3구역
```

### 거점 배치 (확정)

총 8개 (마을형 3 + 간이형 5)

| 위치 | 타입 | 비고 |
|---|---|---|
| 시작점 (1구역 입구) | 마을형 | 게임의 홈 베이스 |
| 1-2 경계 | 간이형 | 초반 |
| 2-3 경계 A | 마을형 | 중반 진입, 스토리 NPC |
| 2-3 경계 B | 간이형 | 복수 진입 통로 |
| 3-4 경계 | 마을형 | 전환점 이후, 진실 관련 NPC |
| 4구역 내부 | 간이형 | 큰 구역 중간 체크포인트 |
| 4-5 경계 | 간이형 | 심부 진입 전 마지막 보급 |
| 5구역 내부 | 간이형 | 최종 보스 전 체크포인트 |

- 후반(4~5구역)에 마을형 없음 → 고립감, 긴장감 강화

### 자연 경계 (초안)

| 경계 | 위치 | 테마 연동 |
|---|---|---|
| 강/개울 | 1-2 경계 | 빛의 숲 → 습지로의 자연스러운 전환 |
| 짙은 안개벽 | 2-3 경계 | 습지 → 폐허. 시야가 끊기는 연출. |
| 무너진 성벽/유적 | 3-4 경계 | 폐허 → 그림자 숲. 왕국의 흔적. |
| 거대 뿌리/절벽 | 4-5 경계 | 그림자 숲 → 심부. 지하로 내려감. |

- 추후 레벨 디자인 단계에서 추가/수정 가능

## 3. 추가조작 유형

### 확정된 규칙 (STAGES.md)
- 각 구역의 특정 스테이지에 숨겨진 추가 조작 존재
- 해당 구역 보스 해금과 무관
- 발견 시 즉각적 혜택, 이후 구역 보스 해금의 필수 조건
- 플레이어가 힌트를 통해 스스로 알아내야 함

### 추가조작 방향 (초안)

| 구역 | 테마 | 추가조작 방향 |
|---|---|---|
| 1구역 | 빛의 숲 | 고대 등불대 점화 → 이후 구역에 빛 경로 생성 |
| 2구역 | 안개 습지 | 수문 조작 → 이후 구역의 물 반사면 활성화 |
| 3구역 | 폐허의 숲 | 봉인 제단 활성화 → 이후 구역 잠금 해제 |
| 4구역 | 그림자 숲 | 그림자 결정 조작 → 5구역 최종 보스실 진입 조건 |

- 초안 수준. 개발 진행하며 구역에 어울리는 안으로 구체화/교체 가능.

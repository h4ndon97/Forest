## REC-RISK-001 — 그림자 회피 가능 → 게임 평면화

- **상태**: OPEN
- **긴급도**: 중
- **영향 영역**: 코어 메카닉
- **레퍼런스 비교**: Don't Starve의 Sanity, Hollow Knight의 Trial 룸

### 우려
플레이어가 그림자 큰 곳을 회피만 하면 **게임이 너무 쉬워짐**. 반대로 그림자 큰 곳에서 강제로 싸워야 하면 **좌절**. 코어 룰의 양극화.

### 레퍼런스 비교
- **Don't Starve Sanity**: 계속 회피 가능하지만, 회피하면 보상도 줄어듦 (꿀, Nightmare Fuel 등 보상이 광기 영역에만)
- **HK 어려움**: 선택적 (Trial/Pantheon 별도 룸)
- 둘 다 "어려움 = 선택"으로 풀고 + 보상으로 유도

### 권장 완화
1. **큰 그림자 영역에서만 드롭하는 자원/재료**를 만들어 회피 = 보상 손실로 균형
   - 예: 큰 그림자 적 처치 시만 Memory Shard 드롭
   - 또는 빛령(REC-CONT-001) 일부를 큰 그림자 영역에 배치
2. **코어 진행 경로는 중간 그림자 강도**로만 강제, 나머지는 선택
3. **성장 시스템 보너스가 그림자 강도에 비례**하도록 (REC-MECH-006 흡혈)

### 의사결정 필요 시점
1구역 밸런싱 검증 시점 (현재 진행 중)

## REC-RISK-002 — 시간 정지 만능 카드 → 코어 자원 평면화

- **상태**: OPEN
- **긴급도**: 중
- **영향 영역**: 코어 메카닉, 보스 디자인
- **레퍼런스 비교**: Sekiro의 패리, Hollow Knight의 Soul

### 우려
시간 자원이 충분하면 모든 보스가 **같은 패턴(정지 → 콤보)으로 풀림**. 1.8 시간 봉인 룸 유무로 양극화될 수 있음. 특히 후반 보스가 다 같은 전략으로 클리어되면 다양성 부족.

### 레퍼런스 비교
- **Sekiro 패리**: 만능이지만 **타이밍이 있음** — 자원 무한이라도 실력 시험
- **HK Soul**: 만능이지만 **회복 vs 공격 트레이드오프** — 자원 한정
- **Don't Starve**: 광기 = 양날의 검 (Shadow Creature 처치 시만 Nightmare Fuel)

### 권장 완화
1. **정지 사용량 누적 시 땅거미 추적 강도 누적** (이미 설계됨, 지속 강화 권장)
2. **일부 보스 = 정지 면역 페이즈 강제** (REC-MECH-009 P3 시간 가속, REC-MECH-010 정지 면역 엘리트)
3. **정지 중에도 적의 그림자는 살아서 위협** (REC-MECH-002 광기 + REC-FX-004 그림자 떨림 결합)
4. **시간 정지 자원 게이지 도입** — 일반 정지에 자원 게이트 적용 (현재 임시 무료)

### 의사결정 필요 시점
Phase 4-0 #3 속성 피니시 거동 결정 시점 또는 그 직후

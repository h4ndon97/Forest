## REC-NARR-001 — 환경 스토리텔링 (정지된 사람들 / 사물)

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: M
- **레퍼런스**: Blasphemous — 자기 채찍질 적 디자인 (적 모습 = 스토리), Death's Door — 보스 룸 도입 환경
- **관련 시스템**: StageSystem, 아트
- **의존**: REC-MECH-001 (Echo Nail) — 회상 메커니즘
- **무효화 조건**: 스토리 전달을 텍스트 위주(컷신/대화)로만 결정 시

### 컨셉
50 스테이지 곳곳에 시간이 정지된 순간의 사람/동물/오브젝트. **Echo Nail로 회상 가능**. 특정 인물 = 같은 인물의 5개 시점 (정지 직전, 봉인 의식, 균열 순간 등). **직접 텍스트 없이 시각만으로 스토리**.

### 레퍼런스 분석
Blasphemous:
- 적 외형 자체가 종교적 죄/회개 표현
- 환경 오브젝트가 lore 매체
- 직접 설명 없이 시각으로만 전달

Death's Door:
- 보스 도입부 환경이 보스 정체 암시
- 보스 클리어 후 환경 변화

### 이 프로젝트 적용
- 정지된 사람들:
  - 도망치다 정지된 사람 (얼굴이 공포)
  - 봉인 의식을 진행하는 제사장들 (5명, 각 손짓 다름)
  - 그림자 왕가 마지막 인물 (앉아서 기다리는 자세)
  - 어린이가 어머니를 부르는 정지 순간
  - 책을 펼친 채 정지된 학자
- 정지된 동물들:
  - 사슴이 도망치는 자세
  - 새가 날다 정지
  - 나무에 매달린 거미 (그림자 왕가 모티프)
- 정지된 오브젝트들:
  - 떨어지다 멈춘 잔
  - 펄럭이다 멈춘 깃발
  - 절단되다 멈춘 나무 (도끼와 함께)

### Echo Nail 회상 텍스트
- 각 정지된 대상 = 1~3줄 짧은 회상
- 일부는 같은 인물의 다른 시점 (5단계 시간 추적)

### 구현 메모
- 각 스테이지에 1~3개 정지된 대상 배치
- StageSystem이 정지된 대상 inventory 보유
- Echo Nail (REC-MECH-001) 트리거 시 회상 .tres 재생

## REC-NARR-002 — NPC 변화 추적 (Quirrel 패턴)

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: M
- **레퍼런스**: Hollow Knight — Quirrel, Cloth (스토리 비트마다 위치/상태 변화)
- **관련 시스템**: NPC, StorySystem, SaveSystem
- **의존**: NPC 시스템 골격 (REC-CONT-007 의존)
- **무효화 조건**: 단순 거점 NPC만 결정 시

### 컨셉
거점 NPC 8명 각자 3~4 단계 변화. **플레이어 행동(어느 보스 처치, Manuscript 몇 장 해독)에 따라 다음 만남 다름**. REC-META-003 다중 엔딩과 결합.

### 레퍼런스 분석
HK Quirrel:
- 첫 만남: 상점 옆에서 잠
- 둘째: City of Tears에서 비를 맞음
- 셋째: Greenpath에서 산책
- 넷째: Blue Lake에서 평온
- 결말: 어느 시점에 보면 Blue Lake에 모자만 남음 (자살 은유)

### 이 프로젝트 적용
- 8 거점 NPC 각자 4 단계
- 변화 트리거:
  - 보스 처치 (5단계 단서)
  - Manuscript 해독 진행도 (REC-CONT-005)
  - Memory Shard 획득량 (REC-MECH-001)
- 결말은 NPC마다 다름:
  - 일부는 진실을 알고 떠남
  - 일부는 정지된 채 발견
  - 일부는 새 거점에서 재회

### 구현 메모
- NPC base 클래스 + 상태 머신
- StorySystem이 NPC 상태 평가 (특정 보스 처치 등 트리거)
- 다이얼로그 .tres에 단계별 분기

## REC-NARR-003 — Lore Trinkets (짧은 시 컬렉션)

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: M
- **레퍼런스**: Blasphemous — Rosary Beads, Hollow Knight — Wanderer's Journal
- **관련 시스템**: InventorySystem, 아트
- **의존**: 작가 협업 (시 작성)
- **무효화 조건**: REC-CONT-005 + REC-NARR-004로 충분히 lore 전달 시

### 컨셉
30~40개 수집 아이템. **각 아이템 = 1~2줄 시 + 픽셀아트**. 메인 스토리와 직접 연결 안 되지만 세계관 깊이. **한국 인디 차별화**.

### 이 프로젝트 적용
- 종류:
  - 자연 모티프 (이끼, 돌, 새의 깃털): 정지된 자연
  - 기물 모티프 (잘린 천, 깨진 잔, 빛바랜 사진): 사람들의 흔적
  - 영적 모티프 (호부, 부적, 작은 우상): 제사장 가문 잔재
- 각자 1~2줄 시 (한국어 시조 풍, REC-MKT-004 시너지)
- 효과:
  - 일부는 단순 lore (스탯 효과 없음)
  - 일부는 미세한 효과 (HP +5, 그림자 강도 +/- 등 — 장신구로 통합)

### 작가 협업
- 30~40 시 = 비교적 작은 협업
- 한국어 작가 (REC-CONT-005와 동일 가능)
- 영문은 후속 패치

### 구현 메모
- ItemRegistry에 새 카테고리 `trinket`
- 인벤토리 메뉴에 Trinket 탭 (장신구와 분리 또는 통합)
- 시 + 일러스트 표시는 REC-UX-003과 같은 UI

## REC-NARR-004 — 컬렉터블 (5권 일기)

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: M
- **레퍼런스**: Blasphemous — 환경 책, Animal Well — 그림 단서
- **관련 시스템**: StorySystem, SaveSystem
- **의존**: 후반 스토리(3~5구역) 확정
- **무효화 조건**: 단일 스토리 전달 매체로 결정 시

### 컨셉
**5권 풀 분량 책**. 빛/그림자 봉인 → 균열 → 정지에 이르는 5세대 제사장의 시점. 물리적으로 5권을 모아야 진엔딩 조건. **메인 스토리 핵심 전달 매체**.

### 이 프로젝트 적용
- 5권 = 5세대 제사장 시점:
  - 1권: 1세대 — 봉인 의식 직전 (이상주의)
  - 2권: 2세대 — 봉인 후 첫 균열 발견 (의심 시작)
  - 3권: 3세대 — 거래의 진실 인식 (자기 부정)
  - 4권: 4세대 — 시간 정지 결정 (자기 희생 = 진실 묻기)
  - 5권: 5세대 (주인공 직전) — 후예에게 남기는 메시지
- 각 책 위치:
  - 1권: 1구역 거점
  - 2권: 2-3 경계
  - 3권: 3구역 깊은 곳
  - 4권: 4구역 비밀 룸
  - 5권: 5구역 직전
- 책 전문 = 1권당 5~10페이지 (긴 텍스트)

### 작가 협업
- 5세대 시점 일관성 = 작가 깊이 협업 필수
- 한국어 1차, 영문 후속

### 구현 메모
- 책 = 특수 collectible (인벤토리와 별도 또는 Manuscript 카테고리 안)
- 메뉴에서 자유 열람
- 5권 모두 모음 = REC-META-003 진엔딩 조건

## REC-NARR-005 — 가족 라인 (5단계 선조)

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: L
- **레퍼런스**: Hollow Knight — Pale King-Hornet 라인 (가족 = 게임 코어 미스터리)
- **관련 시스템**: StorySystem, NPC, BossSystem
- **의존**: 후반 스토리(3~5구역) 확정, REC-MECH-001, REC-NARR-004
- **무효화 조건**: 단일 주인공 스토리로 결정 시

### 컨셉
주인공이 제사장 가문 후예. **게임 진행에 따라 5세대 선조와의 만남**(Echo Nail로 회상). 각자 다른 결정을 내렸음. **"물려받은 것의 무게" 직접 구현**.

### 이 프로젝트 적용
- 5세대 선조와 만남:
  - **1세대 (1구역)**: 봉인 의식 첫 인물 — 이상주의자, "이게 정의롭다고 믿었다"
  - **2세대 (2구역)**: 첫 의심을 품은 자 — "어머니의 결정을 의심하지만 따른다"
  - **3세대 (3구역)**: 진실을 인식한 자 — "거래임을 알았으나 침묵했다"
  - **4세대 (4구역)**: 시간 정지 결정자 — "진실을 묻기 위해 시간을 멈췄다"
  - **5세대 (직전)**: 주인공의 부모 — "후예에게 진실을 알리려 했으나 실패"
- 회상 형식:
  - Echo Nail로 정지된 선조 잔상 보기 (REC-MECH-001 시너지)
  - 일기 페이지(REC-NARR-004) 발견
  - 비밀 보스(REC-CONT-006) = 4세대 또는 1세대 (진실 직면)

### 구현 메모
- 5세대 NPC 디자인 (가족 닮음 표현)
- StorySystem이 5세대 진행도 추적
- 각 세대 발견 시 보상 = Memory Shard + 일기 페이지

## 카테고리 메모

- 5개 모두 후반 스토리(3~5구역) 의존 → **3구역 시작 전에 본 카테고리 결정** 권장
- REC-NARR-001/002/004/005가 서로 강한 시너지 → 한 패키지로 결정 권장
- REC-NARR-003은 독립적, 가장 부담 적음
- 작가 협업이 본 카테고리의 큰 변수 → 한국어 작가 컨택 시점이 결정적
- "물려받은 것의 무게" 테마는 본 카테고리에서 가장 깊이 표현됨

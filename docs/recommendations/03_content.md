## REC-CONT-001 — 빛령 수집 + Light Father NPC

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: M
- **레퍼런스**: Hollow Knight — 46 Grubs + Grubfather 보상 사이클
- **관련 시스템**: StageSystem, GrowthSystem, NPC, SaveSystem
- **의존**: —
- **무효화 조건**: 다른 백트래킹 동기 시스템 채택 시

### 컨셉
50 스테이지 → 최소 30~40 빛령. 시간이 정지되어 잠든 빛의 영혼을 깨운다. 거점 1곳에 "Light Father" NPC 배치, **누적 보상**(영구 스탯/장비/스토리 페이지). 메트로배니아 백트래킹 동기 + 정체성 일치.

### 레퍼런스 분석
HK Grub 시스템:
- 46개 Grub이 맵 곳곳에 갇혀 있음
- 일부는 환경 퍼즐 풀어야 도달
- Grubfather에게 돌려주면 누적 보상 (5개마다, 10개마다, 20개마다)
- 진엔딩 조건 아님 — 순수 컬렉션 동기

### 이 프로젝트 적용
- "잠든 빛령" 스프라이트 (시간 정지로 봉인된 빛 영혼)
- 30~40개를 50 스테이지에 분산
- 일부는 환경 오브젝트 활용 필요 (거울로 빛 비춰야 깨어남 등)
- Light Father NPC: 시작 거점 또는 2-3 경계 거점에 배치
- 보상 누적: 5개마다 영구 스탯, 15개에서 비밀 스킬, 30개에서 장비, 40개에서 스토리 페이지

### 구현 메모
- 빛령 = LightSpirit 엔티티 (BaseEnemy 아님, 단순 인터랙트 오브젝트)
- 깨우는 조건: 등불 빛 접촉 또는 환경 퍼즐
- 깨운 빛령은 SaveSystem에 누적 카운트
- Light Father NPC 다이얼로그에 진행도 표시

## REC-CONT-002 — Cornifer형 측량사 NPC

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: S
- **레퍼런스**: Hollow Knight — Cornifer
- **관련 시스템**: WorldMapUI, NPC, MiniMap
- **의존**: REC-UX-002 (미니맵 단계 시스템)
- **무효화 조건**: 미니맵을 처음부터 풀 가시화하는 정책 채택 시

### 컨셉
사기 전엔 **미니맵이 빈칸/점선**. 측량사를 찾으면 영역 지도 구매. 1구역 = 1 측량사. 거점에서 추가 마커(상점/세이브) 구매. 메트로배니아 표준 + 배움 곡선 자연스러움.

### 레퍼런스 분석
HK Cornifer:
- 영역마다 위치, 흥얼거림으로 위치 노출
- 종이 단서가 화면에 떠 있음
- 사면 그 영역의 지도 base 획득
- 거점 Iselda(아내)에서 마커 추가 구매

### 이 프로젝트 적용
- 1구역마다 1 측량사 배치 (총 5명)
- 측량사 외형: 두건 + 양피지 다발 + 측량 도구 (펜더럼/컴퍼스)
- 가격: 화폐 시스템 미결 → 빛령 또는 Memory Shard로 구매 가능 (REC-MECH-001/REC-CONT-001 연계)
- 거점 NPC가 마커 강화 판매 (체크포인트/상점/숨겨진 통로 마커)

### 구현 메모
- WorldMapUI에 영역별 가시화 플래그 추가
- 측량사 만나면 해당 영역 미니맵 활성화
- 캐릭터 디자인: 5명이 시각적으로 식별 가능해야 (모자 색/장식 차별)

## REC-CONT-003 — 비밀 레이어 (흐르는 시간 영역)

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: L
- **레퍼런스**: Animal Well — 4단계 레이어, Tunic — 보이지 않는 풀이
- **관련 시스템**: StageSystem, TimeSystem
- **의존**: NG+ 결정, REC-META-007 (Inverted Forest)와 중첩 가능
- **무효화 조건**: REC-META-007 채택 시 본 항목 흡수

### 컨셉
1회차 클리어 후 **"시간이 다시 흐르는" 영역 1~2개 해금**. 같은 맵이지만 적 배치/오브젝트/NPC 다른 시간대 버전. 50 스테이지 자산 재활용으로 노력 절감 + 깊이 추가.

### REC-META-007과의 차이
- REC-META-007 (Inverted Forest): **전 영역**을 흐르는 시간 버전으로 재제공 (NG+ 또는 진엔딩 루트)
- REC-CONT-003: **1~2 영역만** 비밀 발견형으로 (1회차 중에도 발견 가능)

### 이 프로젝트 적용
- 1구역 또는 2구역의 1 스테이지를 비밀 진입점으로
- 비밀 진입 = 환경 퍼즐 / 측량사 단서 / Manuscript 해독
- 진입하면 같은 맵의 "시간이 흐르는" 변형 버전
- 보상: 비밀 보스 (REC-CONT-006) 또는 진엔딩 단서

### 구현 메모
- StageSystem에 변형 버전 .tscn 추가 (예: stage_1_2_flowing.tscn)
- 적 배치/오브젝트 다른 .tres
- 진입 트리거가 보존되도록 SaveSystem 통합

- 사망 시 챌린지 시작점으로 (거점 귀환 아님)
- HUD에 웨이브 카운터 + 시련 진행도

## REC-CONT-005 — Manuscript Pages (선조의 일지)

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: M
- **레퍼런스**: Tunic — 매뉴얼 페이지 (Trunic 룬어)
- **관련 시스템**: StorySystem, AbilitySystem, UI
- **의존**: 룬 디자인 선결, 한국어/영어 작가 협업 시점
- **무효화 조건**: 스토리 전달을 컷신 + NPC 대사로만 처리 결정 시

### 컨셉
제사장 가문 후예라는 정체성의 **핵심 전달 매체**. 30~50장 흩어짐. 1차에선 그림만 보임, 빛/그림자 룬을 점진 학습하면 일부 해독되어 비밀 스킬/엔딩 단서 등장. **한국 1인 개발 시장에서 특히 강한 차별화 포인트**.

### 레퍼런스 분석
Tunic Manual:
- 50+ 페이지 매뉴얼 (영문 + Trunic 룬어 혼합)
- 기본 메카닉 안내 + 비밀 단서 + 진엔딩 조건
- 일부 페이지는 룬어로만 = 해독 필요
- 커뮤니티가 협업해서 해독 (Shouldice 의도)

### 이 프로젝트 적용
- 30~50장을 50 스테이지에 분산
- 페이지 종류:
  - **그림 페이지**: 환경 단서 (이 룸은 어떻게 풀지)
  - **룬 페이지**: 빛/그림자 룬으로 작성된 단서
  - **혼합 페이지**: 한국어 + 룬으로 작성, 부분 해독 가능
- 룬 학습:
  - 빛 룬 = 빛 스킬 트리 투자 시 점진 해금
  - 그림자 룬 = 그림자 스킬 트리 투자 시
  - 혼합 룬 = 빛+그림자 둘 다 일정 이상 투자 시
- 30+ 해독 시 비밀 보스(REC-CONT-006), 50/50 시 진엔딩(REC-META-008)

### 구현 메모
- ItemRegistry에 새 카테고리 `manuscript`
- 인벤토리 메뉴에 Manuscript 탭
- 페이지 sprite는 한지 텍스처 + 흑백 룬 (REC-MKT-002 시너지)
- 룬 학습 = AbilitySystem의 별도 플래그 (story.runes.light_lvl_1 등)

### 작가 협업
- 1차 출시: 한국어 우선, 30장 (16장은 1·2구역 + 14장은 후반)
- 후속 패치: 영문 룬, 50장 풀 분량

## REC-CONT-007 — NPC 사이드 라인 (기억 잃은 자)

- **우선순위**: ★★
- **노력**: M
- **레퍼런스**: Hollow Knight — Quirrel, Cloth, Myla (NPC 상태 변화)
- **관련 시스템**: NPC, StorySystem, SaveSystem

거점 8개에 각 1~2명. **시간이 정지된 후 기억을 잃은 자들**. Echo Nail로 그들의 잔상을 보면 짧은 회상 텍스트. 진행도 따라 만남이 변함(잃음/기억 회복/사라짐). 스토리 분산 전달.

### 레퍼런스 분석
HK Quirrel:
- 5~6 단계 만남, 각자 다른 위치
- 대사가 점진 깊어짐
- 결말은 Blue Lake에서 자살(은유)

- NPC 종류:
  - **상점 NPC**: 진행도 따라 시간 정지 이전 회상 단편 추가

- StorySystem이 NPC 상태 트리거
- 다이얼로그 .tres에 단계별 분기

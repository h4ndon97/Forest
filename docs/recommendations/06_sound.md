# 사운드 / 음악 추천

> **STATUS**: 본 문서 모든 항목 PROPOSED. 결정 아님.
> **카테고리 코드**: SND
> **중요**: 현 프로젝트에 **사운드 디자인 문서 자체가 부재**. 본 카테고리 채택 전에 `docs/SOUND.md` 신설 권장.

---

## 사운드 디자인 문서 신설 (선결)

본 카테고리의 모든 항목은 사실상 `docs/SOUND.md` 신설을 전제로 한다. 신설 시 권장 구조:

```
docs/SOUND.md
├── 1. 사운드 디자인 정체성
├── 2. BGM 정책 (구역별 / 거점 / 보스)
├── 3. 환경 음향 (시간 정지 / 흐름 / 광기)
├── 4. SFX 정책 (UI / 전투 / 환경)
├── 5. 동적 음악 시스템 (REC-SND-001)
├── 6. 음향 작가 협업 워크플로우
├── 7. 파일 구조 (audio/ 폴더 정리)
└── 8. 미결 사항
```

`audio/` 폴더는 이미 존재하지만 main_theme.tres만 있는 상태. SOUND.md가 신설되면 본 추천을 바로 적용 가능.

## REC-SND-001 — 동적 음악 레이어 (그림자 강도 반응)

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: M
- **레퍼런스**: Ori — horizontal/vertical adaptive music
- **관련 시스템**: AudioSystem (신설 필요), ShadowSystem
- **의존**: SOUND.md 신설, AudioSystem Autoload 신설
- **무효화 조건**: 작곡가 협업 미확보 시 단일 트랙으로 다운그레이드

### 컨셉
1구역당 1트랙이지만 **4~5 레이어**. 그림자 강도 +1 = 새 레이어 페이드 인. 시간 정지 시 = 모든 레이어 lowpass 필터. EffectsSystem과 연계.

### 레퍼런스 분석
Ori 사운드:
- 지역마다 베이스 트랙 + 보컬/스트링/퍼커션 레이어
- 액션 강도에 따라 레이어 동적 페이드
- 같은 트랙이지만 분위기가 매번 다름

### 이 프로젝트 적용
- 1구역 트랙 = 4 레이어:
  - 베이스: 우드윈드 + 앰비언트
  - L1: 그림자 강도 50%+ → 스트링 추가
  - L2: 그림자 강도 100%+ → 퍼커션 추가
  - L3: 시간 정지 → lowpass 필터 + 베이스만
- 5구역까지 적용 시 = 5 트랙 × 4 레이어 = 20 레이어 (작곡 부담)

### 구현 메모
- AudioSystem Autoload 신설 (BGM 관리 + 레이어 페이드)
- 각 레이어 별도 .ogg 파일 + AudioStreamPlayer
- 페이드는 0.5~1초 (자연스럽게)

### 협업 부담
- 작곡가 협업 필수 (1인 개발 부담 큼)
- 한국 인디 작곡가 컨택 권장

## REC-SND-002 — 정지 음향 (시간 정지의 무음)

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: S
- **레퍼런스**: Ori — 사운드 = 퍼커션 역할
- **관련 시스템**: AudioSystem, TimeSystem
- **의존**: SOUND.md 신설
- **무효화 조건**: —

### 컨셉
시간 정지 시 BGM은 페이드, **발자국/타격음만 남고** + 종이 흔들림/시계 톱니 같은 미세 SFX 추가. **코어 메카닉 사운드 정체성**.

### 이 프로젝트 적용
- 시간 정지 진입: BGM 0.5초 페이드 → 100% lowpass + -20dB
- 정지 중 SFX 풀:
  - 발자국 (강조됨)
  - 타격음 (정상)
  - 미세 SFX: 시계 톱니, 종이 떨림, 먼 곳의 종소리, 그림자 속삭임
- 정지 해제: BGM 0.3초 페이드인 + 미세 SFX 페이드 아웃

### 구현 메모
- AudioSystem이 TimeSystem 상태 변화 구독
- lowpass 필터는 AudioEffect 사용 (Godot 내장)
- 미세 SFX는 짧은 .ogg 5~10종 랜덤 재생

## REC-SND-003 — 보스 음악 페이즈 변화

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: M
- **레퍼런스**: Hollow Knight, Death's Door — 페이즈 음악 변화
- **관련 시스템**: AudioSystem, BossSystem
- **의존**: SOUND.md 신설
- **무효화 조건**: 작곡 부담 우려 시 단일 트랙으로 다운그레이드

### 컨셉
페이즈마다 **동일 멜로디의 변주** (악기 변경). 5보스 × 4페이즈 = 다양화 부담 → **한 보스 메인 모티프 + 페이즈 어레인지 4개**로 절감.

### 이 프로젝트 적용
- 보스마다 메인 모티프 1개 (멜로디 5초 핵심)
- 페이즈마다 어레인지:
  - P1: 베이스 (스트링 위주)
  - P2: 인텐스 (퍼커션 추가)
  - P3: 변형 (다른 키 또는 시간 가속 시 빠르게)
  - P4: 클라이맥스 (모든 악기 + 보컬)
- 동일 모티프이므로 작곡 부담은 보스마다 1개 풀 트랙 + 4 어레인지

### 구현 메모
- BossSystem 페이즈 hook이 AudioSystem에 알림
- 페이즈 전환 시 1초 크로스페이드
- 보스 처치 시 음악 페이드 + 정적

## REC-SND-004 — 거점 음악 (30~40초 우디 루프)

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: S
- **레퍼런스**: Death's Door — 거점, Hollow Knight — Dirtmouth
- **관련 시스템**: AudioSystem
- **의존**: SOUND.md 신설
- **무효화 조건**: —

### 컨셉
거점 8개. **30~40초 루프**, 우디 인스트루먼트(우드윈드/스트링) 만으로. 정체성 = "잠시 시간이 흐르는 안식처". **노력 적고 효과 큼**.

### 이 프로젝트 적용
- 거점 BGM 종류:
  - 마을형 거점 3개: 따뜻한 우디 + 약간의 보컬 험밍
  - 간이형 거점 5개: 미니멀 — 류트 + 환경음
- 트랙 길이 30~40초 (짧지만 자연스러운 루프)
- 거점 진입 = 시간 흐름 → BGM이 자연 시작 (정지 음향 정책 안의 자연스러운 변환)

### 구현 메모
- 8개 거점이지만 마을형/간이형은 같은 트랙 변형 가능 (마을 3개 동일 모티프 변형, 간이 5개 동일)
- 즉 작곡 부담 = 2 트랙 + 변형
- 거점 진입 시 자동 재생 (StageSystem hook)

## REC-SND-005 — 환경 음향 (정지된 자연음)

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: S
- **레퍼런스**: Don't Starve — 분위기 사운드
- **관련 시스템**: AudioSystem
- **의존**: SOUND.md 신설
- **무효화 조건**: —

### 컨셉
새소리, 바람 = 짧은 루프 직후 정지하는 듯한 변형. **"정지된 숲"의 청각 표현**. 일반 자연 음향과 구별되는 정체성.

### 이 프로젝트 적용
- 환경 음향 풀:
  - 새소리: 짧은 트릴 후 끊김 (정지 모티프)
  - 바람: 점차 약해지다 정지
  - 나뭇잎: 서걱거림이 미완성으로 끝남
  - 물소리: 시간 흐름 시만 정상, 정지 시 끊김
- 시간 정지/흐름 상태에 따라 사운드 풀 다름

### 구현 메모
- AudioSystem 환경 음향 채널 (BGM과 별도)
- 각 스테이지가 자기 환경 음향 풀 선언 (.tres)
- 시간 상태에 따라 풀 자동 전환

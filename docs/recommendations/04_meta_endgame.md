## REC-META-001 — 맹세 모드 (Penitences 차용)

- **상태**: PROPOSED
- **우선순위**: ★★★
- **노력**: M
- **레퍼런스**: Blasphemous — Penitences (Bleeding Heart / Unwavering Faith / True Guilt 3종)
- **관련 시스템**: SaveSystem, GrowthSystem, CombatSystem
- **의존**: NG+ 결정
- **무효화 조건**: NG+ 미채택 결정 시

### 컨셉
NG+ 시작 시 3종 맹세 중 1개 선택. **본 게임 정체성에 맞춰 변형**:
- **빛의 맹세**: 시간 정지 사용 불가 (코어 자원 봉인)
- **그림자의 맹세**: 모든 적 = 큰 그림자 (강도 풀 강화)
- **제사장의 맹세**: 사망 시 모든 회상 파편 손실 (REC-MECH-001 의존)

### 레퍼런스 분석
Blasphemous Penitences:
- Bleeding Heart: HP/MP 회복 변경
- Unwavering Faith: 패리/시간 시스템 변경
- True Guilt: 사망 시 페널티 강화
- 각 맹세 클리어 시 고유 보상

### 이 프로젝트 적용
- NG+ 시작 시 맹세 선택 (3종 중 1개 또는 미선택)
- 각 맹세 클리어 시 고유 스킨/장비/엔딩 변주
- 한 번 NG+ 진행 후 다른 맹세로 NG++

### 구현 메모
- SaveSystem에 새 필드 `oath_type`
- 맹세별 modifier가 코어 시스템에 분산 적용
  - 빛의 맹세 → TimeSystem.lock_stop()
  - 그림자의 맹세 → ShadowSystem 강도 항상 max
  - 제사장의 맹세 → 사망 시 SaveSystem.clear_memory_shards()

## REC-META-002 — Pantheon (시간의 사슬 보스 러시)

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: M
- **레퍼런스**: Hollow Knight — Godhome (5 Pantheon, 끝판 = 42 보스 연전)
- **관련 시스템**: BossSystem, StageSystem
- **의존**: 5보스 완성 후 (현재 2/5)
- **무효화 조건**: 끝판 콘텐츠 우선순위 하락 시

### 컨셉
5 보스 + 비밀 보스 1~2 = 6~7 연전. 회복 제한, 사망 시 처음부터. **끝판 콘텐츠**. 추가 Bindings (방어구 봉인/등불 봉인/스킬 봉인) 변주로 코어 메카닉별 시험.

### 레퍼런스 분석
HK Pantheon:
- Pantheon 1~4: 구역별 보스 모음
- Pantheon 5 (Hallownest): 42 보스 연전, 사망 = 처음부터
- Bindings 4종: 영혼/형식/검/체력 (코어 메카닉별 봉인)

### 이 프로젝트 적용
- 1차 출시: Pantheon 1개 (5+1 보스)
- 후속 패치: 4개 Bindings 추가
  - 시간 봉인: 시간 정지 사용 금지
  - 등불 봉인: 등불 강화 -50%
  - 스킬 봉인: 4슬롯 → 1슬롯
  - 그림자 봉인: 그림자 강도 항상 100%

### 구현 메모
- 새 stage_type `PANTHEON`
- 보스 룸 chain (회복은 룸 사이에서만)
- 사망 시 Pantheon 처음으로 (체크포인트 없음)
- 엔트리 NPC가 거점에 배치

## REC-META-004 — Heat / Pact (균열 깊이)

- **상태**: PROPOSED
- **우선순위**: ★★
- **노력**: M
- **레퍼런스**: Hades — Pact of Punishment (16 condition × 1~5 level)
- **관련 시스템**: NG+, GrowthSystem, BossSystem
- **의존**: NG+ 결정
- **무효화 조건**: REC-META-001 맹세 모드로 충분히 변주 충족 시

### 컨셉
NG+에서 균열 깊이 1~10. 각 레벨마다 1~2 modifier 추가 (적 +20%, 시간 자원 -25%, 그림자 영구 강화 등). 보상 = 각 깊이별 처음 클리어 시 고유 장비/도전과제. **깊이 = "물려받은 것의 무게" 메타포**.

### 레퍼런스 분석
Hades Pact:
- 16 condition (적 강화/페널티/보스 페이즈 등)
- 각 1~5 level
- 누적 Heat 점수 = 완료 시 보상
- 20+ Heat에서만 등장하는 무기 보상

### 이 프로젝트 적용
- 균열 깊이 1~10 (작게 시작)
- modifier 8~10종 (조합으로 1~10 깊이 도달)
  - 적 HP +20% / +50%
  - 시간 자원 -25% / -50%
  - 그림자 강도 항상 110% 이상
  - 회복 아이템 효과 -50%
  - 땅거미 이동 속도 +50%
  - 보스 페이즈 추가
  - Memory Shard 획득 -50%
  - 환경 오브젝트 사용 횟수 제한
- 깊이 1, 5, 10 처음 클리어 시 고유 보상

### 구현 메모
- SaveSystem 새 필드 `pact_depth: int`
- modifier 계산은 모든 시스템에 hook
- 메뉴 UI에서 깊이 + modifier 시각화

## REC-META-007 — Inverted Forest (흐르는 숲, NG+ 모드)

- **상태**: PROPOSED (2026-04-26 명분 재정의)
- **우선순위**: ★★
- **노력**: L
- **레퍼런스**: Castlevania SOTN — Inverted Castle
- **관련 시스템**: StageSystem, TimeSystem, NG+
- **의존**: NG+ 결정, REC-META-003 (다중 엔딩 진엔딩 루트)
- **무효화 조건**: NG+ 미채택 결정 시
- **명분 변경 이력**: 2026-04-26 — REC-RISK-003이 "50 풀 양산"으로 RESOLVED되면서 "자산 재활용" 명분이 약화. 대신 **"NG+ 모드에서 흐르는 시간 변형 제공"으로 명분 재정의** — 50 스테이지 풀 양산과 공존 가능한 새 가치 제안.

### 컨셉 (재정의)
50 스테이지가 1차 회차에서는 **시간이 정지된** 상태로 제공된다. NG+ 진입 또는 진엔딩 루트에서는 같은 50 스테이지를 **"시간이 흐르는" 변형 버전**으로 재제공. 적 변형, 식물 자라남, 그림자 왕가 NPC 등장. **자산 재활용이 아닌 NG+ 차별화 콘텐츠**로 가치 제공.

### 레퍼런스 분석
SOTN Inverted Castle:
- 같은 맵을 뒤집어 제공 (실제로는 Y축 flip)
- 적 종류 변경 + 보스 5개 추가
- 게임 분량을 사실상 2배로 (제작 노력은 30%만 추가)
- "Inverted Castle 발견 = 진엔딩 시작" 발견형 즐거움

### 이 프로젝트 적용
- NG+ 진입 시 50 스테이지를 **시간이 흐르는 버전**으로 재제공
- 변경 사항:
  - 식물 자라남 (시간 흐름 결과 = 정지 풀린 후의 숲)
  - 적 변형 (그림자 왕가 잔재가 영향)
  - NPC 변화 (REC-CONT-007과 통합)
  - 일부 룸은 새 환경 퍼즐 추가
  - 새 적 (정지 면역 엘리트, REC-MECH-010 시너지)
- 진입 조건: 5보스 처치 후 NG+ 시작 또는 비밀 진입점 발견 (1차 회차 중 일부 영역만)
- 비밀 보스(REC-CONT-006) 1~2개 추가

### 구현 메모
- 각 스테이지의 "흐르는 버전" .tscn 추가 (sprite 변형 + 적 배치 변경)
- StageSystem이 Inverted/Flowing 모드 플래그 보유
- 새 적 변형 데이터 .tres
- 노력 추정: 50 스테이지 × 30% = 15 스테이지 분량의 작업 (적 배치 변경 + 일부 sprite 변형)
- **명분 재정의 후**: 1차 회차 50 스테이지가 "정지된 숲"으로서 자체 완결성 + NG+가 "흐름 회복된 숲"으로 진엔딩 루트 = **두 회차에 걸친 메타 내러티브** ("물려받은 것의 무게" 테마와 부합)

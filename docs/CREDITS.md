## OFL 1.1 준수 체크리스트 (Steam 출시 시)

- [ ] 게임 빌드에 `LICENSES/Galmuri-OFL.txt`, `LICENSES/Mulmaru-OFL.txt` 포함
- [ ] 게임 내 크레딧/옵션 화면에 폰트명·디자이너·라이선스 명시
- [ ] 폰트 파일명(.ttf) 변경 금지 — "Reserved Font Name" 조항
- [ ] 폰트를 단독으로 판매하지 않음 (OFL 핵심 조항)
- [ ] 폰트를 수정 재배포 시 같은 OFL + 이름 변경

# 시간이 멈춘 숲 — 전체 스테이지 인덱스

> 모든 구역의 스테이지·거점 일람. 각 항목의 잠금/적/기믹/추가조작/구현상태를 한눈에 본다.
> **갱신 규칙**: 새 스테이지가 .tres로 등록되거나 잠금/기믹이 변경되면 본 문서를 즉시 갱신.

## 범례

**구현 상태**
- ✅ 완료 (.tres + .tscn 모두 존재, 플레이 가능)
- ⚙️ 진행 중 (.tres만 존재, .tscn placeholder)
- ⏳ 계획 (문서만)

**잠금 종류** (`StageData.lock_type` enum)
- `NONE` (0) — 잠금 없음
- `LIGHT` (1) — 등불 ON 필요
- `PURIFY` (2) — 특정 스테이지 완전클리어 필요
- `ENVIRONMENT` (3) — `light_sensor:<id>` / `registry:<combo>` / `flag:<id>`
- `ABILITY` (4) — 능력 ID 직접 (예: `light_dash`)

**시간대 표기**: `initial_hour` (0~24, 정수). 6=새벽, 12=정오, 18=황혼, 22=밤.
